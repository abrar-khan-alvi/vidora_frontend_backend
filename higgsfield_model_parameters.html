Higgsfield DoP (Director of Photography)

Animates a still image into a cinematic video clip with 100+ camera motion presets

Required
model required string
Quality tier — choose how fast or polished you want the output to be.
prompt required string
Describe how the scene should move or feel. Be specific — mention camera movement, lighting, wind, emotion.
image_urls required string[]
URL of the source image you want to animate. Should be a clear, well-lit photo.
Optional
motion_id optional UUID
Pick a preset camera movement. Leave empty to let the AI choose based on your prompt. A few popular ones:
motion_strength optional 0 – 1
How intense the movement is. Low = subtle atmosphere. High = dramatic, fast camera work.
subtle dramatic 0.8
seed optional 1 – 1,000,000
Set a number to get the exact same output every time. Useful for testing. Leave random for varied results.
12,345
enhance_prompt optional boolean
Let Higgsfield's AI automatically improve your prompt before generating. Good when you want richer results without rewriting manually.
Enabled — AI will refine your prompt
check_nsfw optional boolean
Blocks inappropriate content from being generated. Keep on for production apps.

Seedance 2.0 by ByteDance

Cinematic multi-shot video with native audio, realistic physics, and character consistency

Required
prompt required string
Describe your video scene. Include subjects, setting, mood, and camera movement for best results.
Optional
input_image optional string (URL)
Provide a starting image to animate from. When set, switches to image-to-video mode.
resolution optional string
Output video quality. Use 720p for drafts and fast previews, 1080p for final delivery.
aspect_ratio optional string
Shape of the output video. 16:9 for landscape/YouTube, 9:16 for Reels/TikTok, 1:1 for square posts.
duration optional integer (seconds)
How long the video should be. Longer clips cost more credits.
4s 15s 5s
seed optional integer
Fix this number to reproduce the exact same output. Change it to get new variations.
42,000
enhance_prompt optional boolean
Automatically rewrites and enriches your prompt before sending to the model for better results.
Disabled — your prompt is used as-is

Kling by Kuaishou

Photorealistic video generation with first-frame to last-frame control and ultra-realism

Model variant
model required string
Which Kling version to use. Newer = better quality but higher credit cost.
Required
prompt required string
Describe the motion, mood, and scene you want. Kling responds well to cinematic language.
Optional
start_image_url optional string (URL)
The first frame of your video. Kling will animate from this image forward.
end_image_url optional string (URL)
The last frame of your video. Kling will interpolate between the start and end image — great for controlled transitions.
duration optional integer (seconds)
Length of the output video in seconds.
5s 10s 5s
aspect_ratio optional string
Aspect ratio of the output. Match this to your target platform.
negative_prompt optional string
Describe what you DON'T want in the video — blurry, distorted faces, bad lighting, etc.
seed optional integer
Reproducibility seed. Same seed = same output every time.
77,777
enhance_prompt optional boolean
Let the AI improve your prompt automatically for richer cinematic output.
Enabled — AI will refine your prompt
