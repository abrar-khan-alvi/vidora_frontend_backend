# 🎬 Vidora AI Studio

**Vidora** is a premium, production-ready AI content creation platform designed for professional creators and studios. It provides a unified, cinematic interface for generating high-fidelity videos, images, and voiceovers using state-of-the-art AI models.

![Vidora Banner](https://images.unsplash.com/photo-1618005182384-a83a8bd57fbe?auto=format&fit=crop&q=80&w=1200&h=400)

## 🚀 Key Features

### 📹 Video Generation Studio
- **Image-to-Video Workflow**: Transform static images into cinematic 4K videos.
- **Studio Dashboard**: Professional player with technical metadata (Resolution, Bitrate, FPS).
- **Pro Exports**: Support for 4K 60FPS output with commercial usage rights.

### 🎙️ VoiceSync AI
- **Dual Synthesis Modes**: 
  - **Read Text**: High-fidelity narration from scripts.
  - **Mimic Voice**: Neural voice cloning for consistent vocal identity.
- **High-Res Audio**: 48kHz Stereo WAV output with mastering-grade quality.

### 🎨 Image Generation Studio
- **Multi-Model Support**: Generate professional artwork in 6+ artistic styles.
- **Variation Engine**: Rapidly iterate on concepts with one-click variation creation.
- **High-Res Export**: Pixel-perfect downloads up to 2048px.

### ⚡ Prompton AI
- **Prompt Engineering**: Built-in studio for optimizing and refining AI prompts for better results.

### 💰 Monetization & Economy
- **Tiered Subscriptions**: Structured pricing ($29 Starter, $97 Creator, $199 Pro).
- **Credit Top-up System**: "PLUS" packs for on-demand usage scaling.
- **Viral Growth**: Automated Free Tier with integrated watermarks.

## 🛠️ Tech Stack

- **Frontend**: React 18, TypeScript, Vite
- **Animations**: Framer Motion (motion/react)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Architecture**: Modular, state-driven "Studio" components

## 📦 Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/abrar-khan-alvi/vidora.git
   cd vidora
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run development server**:
   ```bash
   npm run dev
   ```
